# Current Format
```python
numpy.linspace(start, stop, num = 50 )
```

# With def keyword
```python
def linspace(start, stop, num=50):
```

# Full module path with def
```python
def numpy.linspace(start, stop, num=50):
```

# As assignment/callable
```python
linspace = numpy.linspace(start, stop, num=50)
```
